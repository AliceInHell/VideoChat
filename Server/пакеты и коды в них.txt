Пакет пользователя при входе - 1|name|ip|port
Ответ сервера: 1|id (у пользователя будет свой униальный id) - всё хорошо | 1|0 - пользователь с таким именем уже существует



Пакет пользователя при выходе - 2|id
Ответа сервера нет.



Пакет пользоватя на получение списка друзей - 3|id
Ответ сервера - 3|friendList(отправляются имена пользователей и их id)



Пакет пользователя на звонок абоненту - 4|id_abonent
Ответ сервера : если пользователь принял - 4|1|ip_abonent|port_abonent, 4|0 - пользователь уже говорит с кем-то или отказался



Пакет сервера на звонок пользователю - 5|id_abonent
Оттвет пользователя : 5|1 - если согласен , 5|0 - если нет
